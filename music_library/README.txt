Local music library for the /play_local command
================================================

Place .mp3 or .flac files in this directory (or set MUSIC_LOCAL_DIR to another folder).
You can use subfolders (any depth); /play_local lets you pick a folder first,
then a file inside it via autocomplete.

Only use audio you have the right to play. Those audio files in this folder are
gitignored so they are not committed to the repository.
